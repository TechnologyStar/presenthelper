# 爱国红色答题与集卡奖励系统 文档

## 项目概述

本系统是一个面向公众的爱国主义教育互动平台，围绕答题、签到、集卡、分享与红色故事学习建立正向激励机制，并提供奖励兑换能力。系统采用前后端分离架构，前端以 Vue 3 + Vite 构建，后端以 Express + Sequelize 连接 MySQL。当前集成了 Linux Do OAuth 登录，支持第三方账号一键登录。

**目标用户**: 学习红色文化知识的普通用户，以及负责题库、卡组与奖励管理的运营人员。

**核心价值**:
- 通过答题、签到、集卡等玩法提升学习参与度
- 通过奖励兑换强化学习反馈与持续参与
- 通过管理端能力支持题库与奖励运营
