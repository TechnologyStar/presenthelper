# 集卡与兑换

## 集卡系统

- 卡组由 `CardSet` 定义，包含多张卡片
- 用户收集卡片记录在 `UserCard`
- 卡片稀有度包含 `common`、`rare`、`epic`、`legendary`

## 卡片获取

- 答题合格后随机发放 1 至 3 张卡片
- 每日签到随机发放 1 张卡片
- 连续签到 7 天会提升稀有卡片概率

## 兑换系统

- 兑换类型定义在 `RewardType`
- 兑换码库存保存在 `RewardCode`
- 兑换记录保存在 `RedemptionRecord`

## 兑换规则

- 可通过积分兑换
- 或通过完成指定卡组兑换
- 兑换成功后返回兑换码并生成记录
